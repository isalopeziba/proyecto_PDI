# proyecto_PDI
SafeRoute es una aplicación web que permite clasificar el estado de una calle (Calle Buena, Grietas o Huecos) por medio de una red Neuronal CNN y detectar anomalías mediante un modelo de visión por computadora YOLO. Además, extrae la ubicación de la fotografía mediante metadatos EXIF y la muestra en un mapa interactivo.

Este repositorio integra:

  -Frontend en HTML, CSS y JavaScript
Backend en Python + Flask
✔ Modelo de IA para clasificación y detección
✔ Lectura de ubicación GPS
✔ Visualización de anomalías mediante Canvas
✔ Mapa interactivo con Leaflet
